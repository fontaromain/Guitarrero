Guitarrero is a clone of Frets on Fire™ game.

Main Features :
    Totally free
    Fully written in C++ for best performances
    Fully compliant with Frets On Fire™ songs
    Supports Multiplayer
    Skinnable GUI
    3D Scenery (not yet fully functionnal)
    Multi-language GUI (English, French ...)
	
	
This was a personnal project during school and is not representative of my actual work for the first release. I will try to update the code base during the next month to make it better.	

Website : http://playguitarrero.online.fr/index.php?en
